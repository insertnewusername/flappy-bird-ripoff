# flappy-bird-ripoff
flappy bird
 ripoff flappy bird is a game where gravity is flipped!
 Press space to go down and avoid hitting the pipes

 Every now and then the game has a chance to scroll really fast with pipes spread out more.
 (totally not a bug that I thought was really cool and left there)
